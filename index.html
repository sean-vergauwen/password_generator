About This Project

How does it work?

I built this to generate passwords locally in your browser using cryptographically secure random numbers. No data is ever sent to a server.

Is my data safe?

Absolutely. Your passwords never leave your device. I don't have a backend, and I'll never track what you generate.

Why did I make this?

I wanted a clean, fast, and private way to create strong passwords without trusting third-party managers for every little thing.

Open Source

This is a personal project, and the code is fully open. You can audit it yourself on GitHub to verify how I handle your security.
